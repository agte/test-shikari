# Запуск
Запустить в консоли
```sh
docker-compose up
```
Затем открыть в браузере http://localhost:3000.

Если порт 3000 на машине уже занят, можно ззадать любой другой порт через переменную окружения, например 5000:
```sh
export PORT=5000
docker-compose up
```
И открыть в браузере http://localhost:5000.
